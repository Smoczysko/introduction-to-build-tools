# Introduction to build tools

This workspace consists build tools (Ant, Maven, Gradle) usage examples for series of lectures at Gda≈Ñsk University of Technology. They are categorized in different directories, one for each tool usage example.

## Structure ##

Each example has it's own directory (independent Ant/Maven/Gradle project). This directory in named in <example_number>-<build_tool>-<topic> pattern.

## Requirements ##

This project has very small system requirements:

* Java SE SDK 1.7 or higher
* Apache Maven 3.0 or higher
